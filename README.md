## Running locally

Clone repository:

```
git clone https://github.com/endormi/www.git
```

Get submodule `hugo-coder`:

```
git submodule init
```

Run website:

> For Linux

```
make run-local
```

> For Windows

```
cd scripts && run.bat
```
